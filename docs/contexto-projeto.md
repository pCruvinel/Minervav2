# Contexto do Projeto Minerva v2

## Visão Geral
O **Minerva v2** é uma aplicação web focada na gestão de Ordens de Serviço (OS), Financeiro e Dashboard, construída com tecnologias modernas para garantir performance e escalabilidade. O projeto está em fase de migração e unificação de componentes, com foco na melhoria da experiência do usuário e na robustez do sistema.

## Stack Tecnológico
- **Frontend:** React, Vite, TypeScript
- **Estilização:** Tailwind CSS, Shadcn UI (Radix UI)
- **Backend/Banco de Dados:** Supabase (PostgreSQL)
- **Gerenciamento de Estado:** React Query (provável), Context API
- **Deploy:** Vercel (provável)

## Estrutura de Documentação
A documentação do projeto foi centralizada na pasta `docs/` para facilitar o acesso e a manutenção.

- **[Planejamento](docs/planning/):** Documentos de arquitetura, planos de implementação, checklists e relatórios de status.
- **[Guias](docs/guides/):** Manuais de uso, guias de início rápido (Quick Start) e instruções para colaboradores.
- **[Técnico](docs/technical/):** Documentação de API, esquemas de banco de dados, guias de integração e solução de problemas (troubleshooting).

## Módulos Principais
- **OS (Ordens de Serviço):** Gestão completa do ciclo de vida das OS, com workflows personalizados.
- **Financeiro:** Controle de contas a pagar/receber e relatórios financeiros.
- **Dashboard:** Visão geral dos indicadores de desempenho.
- **Autenticação:** Sistema de login e controle de acesso via Supabase Auth.
